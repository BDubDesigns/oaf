# OAF Stack 0.1

The blessed app stack is Next.js with TypeScript and pnpm; Postgres with
Drizzle; Better Auth; Zod; Tailwind CSS; Vitest; Playwright; and Docker /
Coolify-ready deployment artifacts.

Apps are generated from pinned stack snapshots. Do not add `@latest`, floating
direct dependency ranges, or a new framework choice. This fixture/skeleton has
no installed dependencies yet, so do not assume a real Next.js runtime exists.
